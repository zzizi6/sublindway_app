## 설명
- 카카오맵 api를 사용하여 유저의 승차 하차 위치를 표시
- 백엔드와 통신하여 유저의 열차 위치를 노선도에 실시간으로 표시 (실시간 열차 이동)
- 유저의 승차사진과 객체인식사진을 각 탭에 저장
- 반응형 웹 구현을 통해 웹 뿐만 아니라 모바일에서도 확인 가능
